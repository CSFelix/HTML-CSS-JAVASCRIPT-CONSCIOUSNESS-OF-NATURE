RECICLAR: PARA QUE?

Quantas vezes você já descartou algum objeto sem utilidade considerando como lixo? Neste blog vamos lhe mostrar alternativas criativas para reaproveitar o lixo de maneira útil.


PAPEL

Com rolos de papel higiênico é possível fazer uma decoração de parede em forma de flor, tudo o que você precisa é de vários rolos, tesoura e cola quente.


PLÁSTICO

Com colheres de plástico é possível fazer uma decoração de mesa em forma de flor , tudo que precisamos é de colheres de plástico, um copo , fita ou podemos fazer com cola.


VIDRO

Com um pote de vidro é possível fazer uma linda decoração , usando apenas o pote e uma foto que deseja colocar dentro. Podemos tambem colorir o vidro.


PNEU

Com pneus, é possível fazer um pequeno pula-pula para as crianças. Elas irão adorar.


METAL

Com múltiplas engrenagens, podemos criar um suporte para abajures. Assim, quando seu abajur quebrar, não descarte-o, use sua imaginação☺.


MADEIRA

Que tal fazer um vaso com pregadores para suas plantas? Além de ser econômico, deixa suas flores mais belas.
